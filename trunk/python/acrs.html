Automatic classless route summarization (ACRS) web demo

ACRS is an open source library that can be used to summarize lists of IPv4 prefixes or routes classlessly. This page hosts a demonstration program that uses the ACRS library.

Instructions:

    1. In the box below, enter a list of IPv4 prefixes in CIDR notation, separated by spaces or newlines (e.g. 192.168.1.0/25 192.168.1.128/25)
    2. Click the Summarize button
See acrs.googlecode.com for more information, including source code for the ACRS library in python and C++ and a command line version of this demo.
